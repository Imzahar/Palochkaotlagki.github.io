О "Палочке отладки"

Данный инструмент предназначен для быстрого доступа к тестовым функциям и загрузке необходимых ресурсов. Все кнопки ведут на Google.com с соответствующими поисковыми запросами.

Используйте тестовые кнопки для проверки различных сценариев, а кнопки скачивания — для получения дополнительных материалов по работе с инструментом.

Как изменить сортировку кнопок

Для изменения порядка отображения кнопок найдите в CSS раздел "НАСТРОЙКИ СОРТИРОВКИ" и измените значения свойства "order" для соответствующих классов (.btn-test-1, .btn-test-2, ... и .btn-download-1, .btn-download-2, ...).

Элементы с меньшим значением order отображаются раньше. Текущая настройка: сортировка по возрастанию (от 1 к 15 и от 1 к 10).
